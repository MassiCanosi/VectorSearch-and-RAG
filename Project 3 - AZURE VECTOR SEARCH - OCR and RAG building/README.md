# ChatBotCompliance

## Overview

The Compliance Chatbot is an AI-powered assistant designed to help businesses and organizations manage regulatory compliance efficiently.
It provides automated responses to compliance-related queries, helps navigate legal and policy frameworks, and ensures adherence to industry-specific regulations.

## Features

Natural Language Processing (NLP): Understands and responds to compliance-related inquiries.


Customizable Knowledge Base: Supports organization-specific compliance policies.

## Technologies Used

OCR: Python (OpenCV, Tesseract)

NLP Model: OpenAI GPT

Frontend: HTML, CSS. Poster on Company's Sharepoint

Deployment: Azure
